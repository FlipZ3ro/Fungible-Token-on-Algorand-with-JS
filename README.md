# Fungible-Token-on-Algorand-with-JS
## Installation
```
npm init --yes
npm install algosdk
npm list algosdk
```

## Create Wallet
```
node CreateAccount.js
```

## Fetch API key
Create an account on PureStake: https://developer.purestake.io/login

## Deploy a Fungible Token
```
node CreateAsset.js
```
